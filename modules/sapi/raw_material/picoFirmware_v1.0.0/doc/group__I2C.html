Detailed Description

Inter-Integrated Circuit communication API

Available signals and pins:

picoCIAA Signal
P8_5 I2C1_SCL
P8_8 I2C1_SDA
P8_3 I2C0_SCL
P8_6 I2C0_SDA
P8_7 I2C2_SCL
P2_13 I2C2_SDA
Note
use I2C1 as default because it has external pull-ups available
